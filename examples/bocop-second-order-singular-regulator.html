Second order singular regulator

rem: Second order singular regulator

We consider a second order singular regulator problem:

\[\min \int_0^T (x^2(t) + \dot x^2(t)) \ \mathrm{d}t; \quad \dot x(t) = u(t) \in [-1,1].\]

The difference with Fuller’s problem is that the cost function includes a penalization of the “speed” \(\ddot x(t)\). We observe in the solution the occurrence of a singular arc, the optimal control being of the form bang (-1) - singular.

References:

  1. G.M. Aly, The computation of optimal singular control, International Journal of Control, 28(5):681-688, 1978.
  2. S. Aronna, F.J. Bonnans and P. Martinon, A well-posed shooting algorithm for optimal control problems with singular arcs, Journal of Optimization Theory and Applications, 2013. Online First, January 2013.

See: BOCOP - The optimal control solver. 

par:
  tf = 5

dyn:
  x1 x2 u j

lim:
  -1 <= u <= 1
  
t=t0:
  x1 = 0
  x2 = 1
  j  = 0
 
ini:
  x1 = 0
  x2 = 1
  u = -1/tf
  j = 0
  
equ:
  x1´ == x2
  x2´ == u
  j´  == x1*x1 + x2*x2
  
obj:
  minimize final(j) using nlp
