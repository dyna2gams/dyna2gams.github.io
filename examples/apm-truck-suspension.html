Truck suspension

rem: Truck suspension

Reference:

  1. D. Matko, R. Karba and B. Zupancic, Simulation and Modeling of Continuous Systems, Prentice Hall, 1992.

See: APMonitor Modeling Language 

par:
  tf = 3
  m1 = 500     # kg
  m2 = 50      # kg
  k1 = 7500    # spring constant
  k2 = 150000  # spring constant
  f = 2250     # N
  z = 0        # vertical displacement

dyn:
  y1 y2  # position
  z1 z2  # velocity

t=t0:
  y1 = -0.05 
  y2 = 0
  z1 = 0     
  z2 = 0

ini:
  y1 = -0.05 
  y2 = 0
  z1 = 0     
  z2 = 0

equ:
  y1´ == z1
  y2´ == z2
  m1 * z1´ == -f * (z1-z2) - k1 * (y1-y2)
  m2 * z2´ == -f * (z2-z1) - k1 * (y2-y1) - k2 * (y2-z)

obj:
  maximize final(y1) using lp
