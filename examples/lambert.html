A simple Lambert problem

rem: A simple Lambert problem

A simple Lambert problem transcribed as a constraint satisfaction problem. The initial guess determines the multiple revolution selected.

Source: Dario Izzo (Nov 2006) - https://github.com/esa/SpaceAMPL 

par:
  tf = 20

dyn:
  x{1:6}

t=t0:
  x{1:3} = {1 0 0}

t=tf:
  x{1:3} = {0 1 0}

equ:
  x1´ == x4
  x2´ == x5
  x3´ == x6
  x4´ == -x1/(sqr(x1)+sqr(x2)+sqr(x3))^(3/2)
  x5´ == -x2/(sqr(x1)+sqr(x2)+sqr(x3))^(3/2)
  x6´ == -x3/(sqr(x1)+sqr(x2)+sqr(x3))^(3/2)

ini:
  x1 = cos(linspace(0,tf))
  x2 = sin(linspace(0,tf))
  x4 = -sin(linspace(0,tf))
  x5 = cos(linspace(0,tf))

obj:
  minimize 0 using nlp
