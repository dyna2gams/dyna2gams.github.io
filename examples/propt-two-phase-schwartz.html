Two-Phase Schwartz

rem: Two-Phase Schwartz

Find \(u\) over \(t \in [0,2.9]\) to minimize \[ J = 5 \ (x_1(t_f)^2 + x_2(t_f)^2) \] subject to: \[ \dot{x}_1 = x_2 \\ \dot{x}_2 = u - 0.1 \ (1+2 \ x_1^2) \ x_2 \\ x(0) = [1 \ 1] \] and path constraints for \(t < 1\): \[ 1-9 \ (x_1-1)^2-(\frac{x_2-0.4}{0.3})^2 \leq 0 \\ -0.8 \leq x_2 \quad (t < 1) \\ -1 \leq u \leq 1 \quad (t < 1) \]

References:

  1. M. &#268;ižniar, M. Fikar, M. A. Latifi, MATLAB™ Dynamic Optimisation Code DYNOPT User’s Guide, Technical Report, KIRP FCHPT STU Bratislava, Slovak Republic, 2006.
  2. Schwartz, A. L., Theory and Implementation of Numerical Methods based on Runge-Kutta Integration for Solving Optimal Control Problems, Ph.D. Dissertation, University of California, Berkeley, 1989.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010 

set:
  phase = ph1:ph2

par:
  tf = 1
  tlen[phase] = [ph1 1, ph2 1.9]

dyn:
  t[phase]
  x{1:2}[phase]
  u[phase] :: control

lim:
  -0.8 <= x2['ph1'] <= +inf
  -1 <= u['ph1'] <= 1
  -50 <= u['ph2'] <= 50

t=t0:
  t['ph1'] = 0
  x{1:2}['ph1'] = 1

ini:
  x{1:2}[phase] = 1
  u[phase] = 0

exp:
  TimeDot == tlen[phase] / tf

equ:
   t´[phase] / TimeDot == 1
  x1´[phase] / TimeDot == x2[phase]
  x2´[phase] / TimeDot == u[phase] - 0.1*(1+2*sqr(x1[phase]))*x2[phase]
  9*sqr(x1['ph1']-1) + sqr((x2['ph1']-0.4)/0.3) >= 1
  @linkphases

obj:
  minimize 5*sqr(final(x1['ph2']))+sqr(final(x2['ph2'])) using nlp
