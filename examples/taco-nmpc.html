Simple nonlinear model predictive control toy problem

rem: Simple nonlinear model predictive control toy problem

Simple nonlinear model predictive control (NMPC) toy problem with unstable steady state.

Christian Kirches, Sven Leyffer, July 2011

Reference:

  1. M. Diehl, Real-Time Optimization for Large Scale Nonlinear Processes, PhD thesis, Universität Heidelberg, 2001.

See: mintoc.de 

par: p = 1

dyn: x j u

var: tf

lim:
  2 <= tf <= 3
  -1 <= u <= 1

t=t0:
  x = -0.5
  j = 0

t=tf:
  x = 0

ini:
  tf = 3
  x = -0.5
  u = 0

equ:
  j´ == x*x + u*u
  x´ == (x+p)*x + u

obj: minimize final(j) using nlp
