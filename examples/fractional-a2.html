A simple fractional order control problem

rem: A simple fractional order control problem

Reference:

  1. N.H. Sweilam, T.M. Al-Ajami and R.H.W. Hoppe, Numerical Solution of Some Types of Fractional Optimal Control Problems, Hindawi Publishing Corporation, The Scientific World Journal, Volume 2013, Article ID 306237 

par:
  gam = 0.7
  tf = 1

gms:
  $$libinclude dyna-fractder JG gam

dyn:
  x u wx wu
  J

t=t0:
  x = 0
  J = 0

t=tf:
  x = 2/gamma(gam+3)

equ:
  x´ == u + Time^2 - fractder(x,gam)
  J´ == sqr(Time*u-(gam+2)*x)
  wx == 2*Time^(gam+2)/gamma(gam+3)	# the optimal solution
  wu == 2*Time^(gam+1)/gamma(gam+2)	# the optimal solution
  
obj:
  minimize final(J) using nlp
