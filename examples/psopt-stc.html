Time varying state constraint problem

rem: Time varying state constraint problem

Source: PSOPT - Optimal Control Solver, User Manual, Release 4 build 2015. 

par:
  tf = 1

dyn:
  x{1:2}   # state variables
  u        # control variable
  J        # cost criterion
  z        # path constraint
  
lim:
  -2 <= x{1:2} <= 2
  -20 <= u <= 20
  -100 <= z <= 0
  
t=t0:
  x1 = 0
  x2 = -1
  J = 0

equ:
  x1´ == x2
  x2´ == -x2 + u
  z == -8*sqr(Time-0.5) + 0.5 + x2 
  J´ == x1*x1 + x2*x2 + 0.005*u*u

obj:
  minimize final(J) using nlp
