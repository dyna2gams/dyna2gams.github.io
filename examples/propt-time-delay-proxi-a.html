Time Delay (Approximate)

rem: Time Delay (Approximate)

Find \(u\) over \(t \in [0,5]\) to minimize \[ J = x_3(t_f) \] subject to: \[ \dot{x}_1 = x_2 \\ \dot{x}_2 = -10 x_1-5 x_2-2 x_1(t-\tau)-x_2(t-\tau)+u \\ \dot{x}_3 = 0.5 (10 x_1^2+x_2^2+u^2) \\ \tau = 0.25 \] A Taylor series expansion gives: \[ \dot{x}_2 \approx (-12 x_1+(2 \tau-6) x_2 + u)/(1-\tau) \] The initial conditions are: \[ x(0) = [1 \ 1 \ 0] \\ -\infty \leq u \leq +\infty \]

Reference:

  1. R. Luus, Iterative Dynamic Programming, CHAPMAN & HALL/CRC Monographs and Surveys in Pure and Applied Mathematics.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010. 

par:
  tau = 0.25
  tf = 5

dyn:
  x1 x2 x3	# state variables
  u	# control variable

t=t0:
  x1 = 1
  x2 = 1
  x3 = 0

ini:
  x1 = 1
  x2 = 1
  x3 = 0
  u = 0

equ:
  x1´ == x2
  x2´ == (-12*x1+(2*tau-6)*x2 + u)/(1-tau)
  x3´ == 0.5*(10*sqr(x1)+sqr(x2)+sqr(u))

obj:
  minimize final(x3) using nlp
