Hyper-Sensitive Problem

rem: Hyper-Sensitive Problem

This example is taken from the following reference:

  1. A.V. Rao and K.D. Mease, Eigenvector Approximate Dichotomic Basis Methods for Solving Hyper-Sensitive Optimal Control Problems, Optimal Control Applications and Methods, 21(1):1-17, January-February 2000. 

par:
  tf = 5000

dyn:
  x u j

lim:
  -50 <= x <= 50
  -50 <= u <= 50

t=t0:
  x = 1.5
  j = 0

t=tf:
  x = 1

ini:
  x = linspace(initial(x),final(x))
  u = 0

equ:
  x´ == -x*x*x + u
  j´ == 0.5*(x*x +u*u)

obj:
  minimize final(j) using nlp
