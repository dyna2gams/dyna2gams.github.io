Linear Tangent Steering Problem

rem: Linear Tangent Steering Problem

Find \(u(t)\) over \(t \in [0,t_f]\) to minimize \[ J = t_f \] subject to: \[ \ddot{y}_1 = a \ \cos(u) \\ \ddot{y}_2 = a \ \sin(u) \\ |u| \leq \frac{\pi}{2} \\ y_{1:2}(0) = 0 \\ \dot{y}_{1:2}(0) = 0 \\ a = 1 \\ y_{2}(t_f) = 5 \\ \dot{y}_{1:2}(t_f) = [45 \ 0] \] The following transformation \[ x_1 = y_1 \\ x_2 = \dot{y}_1 \\ x_3 = y_2 \\ x_4 = \dot{y}_2 \] gives a new formulation: \[ \dot{x}_1 = x_2 \\ \dot{x}_2 = a \ \cos(u) \\ \dot{x}_3 = x_4 \\ \dot{x}_4 = a \ \sin(u) \]

Reference:

  1. E. D. Dolan, J. J. Moré and T. S. Munson, Benchmarking Optimization Software with COPS 3.0, Argonne National Laboratory, Technical Report ANL/MCS-TM-273, February 2004.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010 

par:
  a = 100

var:
  tf

dyn:
  x{1:4}	# state variables
  u		# control variable

lim:
  0 <= tf <= 10 
  -pi/2 <= u <= pi/2

t=t0:
  x{1:4} = 0

t=tf:
  x2 = 45
  x3 = 5
  x4 = 0

ini:
  tf = 1
  x1 = linspace(0,12)
  x2 = linspace(0,45)
  x3 = linspace(0,5)
  x4 = 0
  u = 1

equ:
  x1´ == x2
  x2´ == a*cos(u)
  x3´ == x4
  x4´ == a*sin(u)

obj:
  minimize tf using nlp
