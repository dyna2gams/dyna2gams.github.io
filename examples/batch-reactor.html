Batch reactor

rem: Batch reactor

This batch reactor problem describes the consecutive reaction of some substance A via substance B into a desired product C. The system is interacted with via the control function \(u(t)\) which stands for the temperature. The goal is to produce as much of substance B (which can then be converted into product C) as possible within the time limit.

Reference:

  1. J.G. Renfro, A.M. Morshedi, O.A. Osbjornsen, Simultaneous optimization and solution of systems described by differential/algebraic equations, Computers and Chemical Engineering, 11:503-517, 1987.

See: mintoc.de

The optimal objective value of the problem is \(x_2(t_f) = -0.611715\) 

var:
  tf

dyn:
  x1  # represent the concentration of component A
  x2  # represent the concentration of component B
  u   # the temperature of reactor (control variable)

lim:
  0 <= tf <= 1
  298 <= u <= 398

t=t0:
  x1 = 1
  x2 = 0

ini:
  tf = 1
  x1 = 0
  x2 = 0
  u = 350

exp:
  k1 == 4000 * exp(-2500/u)
  k2 == 620000 * exp(-5000/u)

equ:
  x1´ == -k1*x1*x1
  x2´ == k1*x1*x1 - k2*x2

obj:
  maximize final(x2) using nlp
