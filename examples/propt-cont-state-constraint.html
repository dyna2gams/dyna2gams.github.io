Continuous State Constraint Problem

rem: Continuous State Constraint Problem

Find \(u(t)\) over \(t \in [0,1]\) to minimize \[ J = \int_0^1 (x_1^2(t) + x_2^2(t) + 0.005 \ u^2(t)) \ \mathrm{d}t \] subject to: \[ \dot{x}_1 = x_2 \\ \dot{x}_2 = -x_2+u \\ x_1(0) = 0 \\ x_2(0) = -1 \\ 8 \ (t-0.5)^2 - 0.5 - x_2 \geq 0 \]

Reference:

  1. L.S. Jennings, M.E. Fisher, K.L. Teo, C.J. Goh, MISER3 Optimal Control Software: Theory and User Manual Version 2.0, Department of Mathematics, The University of Western Australia, Nedlands, WA 6907, Australia, 1997.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010. 

par:
  tf = 1

dyn:
  x1 x2	# state variables
  u	# control variable
  J

lim:
  -10 <= x1 <= 10
  -10 <= x2 <= 10
  -20 <= u <= 20

t=t0:
  x1 = 0
  x2 = -1
  J = 0
  
ini:
  x1 = 0
  x2 = -1

equ:
  x1´ == x2
  x2´ == -x2+u
  8*sqr(Time-0.5) >= 0.5+x2
  J´ == sqr(x1) + sqr(x2) + 0.005*sqr(u)

obj:
  minimize final(J) using nlp
