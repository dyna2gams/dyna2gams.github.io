A singular control problem

rem: A singular control problem

This example is an established benchmark problem commonly referred to as “singular control problem” and considered for global optimal control. It is given in [1] in which the authors propose to solve global optimal control problems with a new algorithm that is based on Bock’s direct multiple shooting method.

Reference:

  1. H. Diedam and S. Sager, Global optimal control with the direct multiple shooting method, Optimal Control Applications and Methods, 00:1-22, 2016. 

par:
  tf = 1
  
dyn:
  x{1:5}
  u

lim:
  -4 <= u <= 10

t=t0:
  x{1:5} = {0 -1 -sqrt(5) 0 0}
  
equ:
  x1´ == x2
  x2´ == -x3*u + 16*Time - 8
  x3´ == u
  x4´ == sqr(x1) + sqr(x2) + 5e-4*sqr(x2+16*x5-8-x3*u*u/10)
  x5´ == 1
  u :: constant(4)
  
obj:
  minimize final(x4) using nlp
