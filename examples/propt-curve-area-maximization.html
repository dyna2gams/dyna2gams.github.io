Curve Area Maximization

rem: Curve Area Maximization

Maximal area under a curve of given length.

Find \(u\) over \(t \in [0,1]\) to minimize: \[ J = \int_0^{1} x_1 \ \mathrm{d}t \] subject to: \[ \dot{x}_1 = u \\ \dot{x}_2 = \sqrt{1+u^2} \\ x(t_0) = [0 \ 0] \\ x(t_f) = [0 \ \frac{\pi}{3}] \]

Reference:

  1. I. Ioslovich and P.-O. Gutman, On smooth optimal control determination, Automatica, 40(12):2175-2180, 2004.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010. 

par:
  tf = 1

dyn:
  x1 x2	# state variables
  u	# control variable
  J

t=t0:
  x1 = 0
  x2 = 0
  J = 0

t=tf:
  x1 = 0
  x2 = pi/3

ini:
  x1 = 1
  x2 = linspace(0,pi/3)

equ:
  x1´ == u
  x2´ == sqrt(1+sqr(u))
  J´ == x1

obj:
  maximize final(J) using nlp
