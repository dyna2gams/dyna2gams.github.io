Genetic

rem: Genetic

Find \(u\) over \(t \in [0,1]\) to minimize \[ J = \int_0^{1} u^2 \ \mathrm{d}t \] subject to: \[ \dot{x} = x^2 + u \] The initial conditions are: \[ x(0) = 0 \\ x(1) = 0.5 \]

Reference:

  1. O.S. Fard and A.H. Borzabadi, Optimal Control Problem, Quasi-Assignment Problem and Genetic Algorithm, Proceedings of World Academy of Science, Engineering and Technology Volume 21 January 2007 ISSN 1307-6884.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010. 

par:
  tf = 1

dyn:
  x	# state variables
  u	# control variable
  J

t=t0:
  x = 0
  J = 0

t=tf:
  x = 0.5

ini:
  x = 0
  u = 0

equ:
  x´ == sqr(x)+u
  J´ == sqr(u)

obj:
  minimize final(J) using nlp
