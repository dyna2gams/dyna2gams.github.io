Zermelo Flight Trajectory

rem: Zermelo Flight Trajectory

Time-optimal aircraft heading through air in motion.

Reference:

  1. Bryson, A. E. and Ho, Y.-C., Applied Optimal Control, Hemisphere Publishing, New York, 1975, p. 77.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010 

par:
  v = 1

var:
  tf

dyn:
  x1 x2	# state variables
  u	# control variables

lim:
  1 <= tf <= 10

t=t0:
  x1 = -2
  x2 = 0

t=tf:
  x1 = 0.5
  x2 = -1.6

ini:
  x1 = -2
  x2 = 0
  u = pi

exp:
  wh == exp(-x1*x1-x2*x2+0.25)

equ:
  x1´ == v*cos(u)+x2*wh
  x2´ == v*sin(u)-x1*wh

obj:
  minimize tf using nlp
