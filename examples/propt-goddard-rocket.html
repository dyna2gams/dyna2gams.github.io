Goddard Rocket, Maximum Ascent

rem: Goddard Rocket, Maximum Ascent

Find \(u(t)\) over \(t \in [0,t_f]\) to minimize \[J = h(t_f)\] subject to: \[ \dot{v} = \frac{1}{m}(t_f-D)-g \\ \dot{h} = v \\ \dot{m} = -\frac{t_f}{c} \\ D = D_0 v^2 \exp(-\beta \frac{h-h_0}{h_0}) \\ g = g_0 (\frac{h_0}{h})^{2} \\ m(0) = 1 \\ m(t_f) = 0.6 \\ v(0) = 0 \\ h(0) = 1 \\ g_0 = 1 \\ 0 \leq u \leq 3.5 \\ D_0 = 0.5 \times 620 \\ c = 0.5 \\ \beta = 500 \]

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010. 

par:
  beta = 500
  c = 0.5

var:
  tf

dyn:
  v h m		# state variables
  T		# control variable

lim:
  0.1 <= tf <= 1
  0 <= v <= +inf
  1 <= h <= +inf
  0.6 <= m <= 1
  0 <= T <= 3.5

t=t0:
  v = 0
  h = 1
  m = 1

t=tf:
  m = 0.6

ini:
  tf = 1
  v = 620
  h = 1
  m = linspace(initial(m),final(m))
  T = 0

exp:
  D == 0.5*620*sqr(v)*exp(-beta*h)
  g == 1/sqr(h)

equ:
  v´ == (T-D)/m - g
  h´ == v
  m´ == -T/c

obj:
  maximize final(h) using nlp
