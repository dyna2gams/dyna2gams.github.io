Two-Link Robotic Arm

rem: Two-Link Robotic Arm

Find \(u\) over \(t \in [0,t_f]\) to minimize \[ J = t_f \] subject to: \[ \dot{x}_1 = \frac{\sin(x_3) (\frac{9}{4} \cos(x_3) x_1^2+2 x_2^2) + \frac{4}{3} (u_1-u_2) - \frac{3}{2} \cos(x_3) u_2 } {\frac{31}{36} + \frac{9}{4} \sin^2(x_3)} \\ \dot{x}_2 = -\frac{\sin(x_3) (\frac{7}{2} x_1^2+\frac{9}{4} \cos(x3) x_2^2) - \frac{7}{3} u_2 + \frac{3}{2} \cos(x_3) (u_1-u_2) }{\frac{31}{36} + \frac{9}{4} \sin^2(x_3)} \\ \dot{x}_3 = x_2-x_1 \\ \dot{x}_4 = x_1 \] The boundary conditions are: \[ x(0) = [0 \ 0 \ 0.5 \ 0] \\ x(t_f) = [0 \ 0 \ 0.5 \ 0.522] \\ -1 \leq u_1 \leq 1 -1 \leq u_2 \leq 1 \]

Reference:

  1. R. Luus, Iterative Dynamic Programming, CHAPMAN & HALL/CRC Monographs and Surveys in Pure and Applied Mathematics.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010. 

var:
  tf

dyn:
  x1 x2 x3 x4	# state variables
  u1 u2	# control variables

lim:
  2.6 <= tf <= 100
  -1 <= u1 <= 1
  -1 <= u2 <= 1

t=t0:
  x1 = 0
  x2 = 0
  x3 = 0.5
  x4 = 0

t=tf:
  x1 = 0
  x2 = 0
  x3 = 0.5
  x4 = 0.522

ini:
  tf = 3
  x1 = 0
  x2 = 0
  x3 = 0.5
  x4 = 0.522
  u1 = linspace(1,-1)
  u2 = linspace(1,-1)

equ:
  x1´ == (sin(x3)*(9/4*cos(x3)*sqr(x1)+2*sqr(x2)) + 4/3*(u1-u2) - 3/2*cos(x3)*u2 ) / (31/36 + 9/4*sqr(sin(x3)))
  x2´ == -(sin(x3)*(7/2*sqr(x1) + 9/4*cos(x3)*sqr(x2)) - 7/3*u2 + 3/2*cos(x3)*(u1-u2)) / (31/36 + 9/4*sqr(sin(x3)))
  x3´ == x2 - x1
  x4´ == x1

obj:
  minimize tf using nlp
