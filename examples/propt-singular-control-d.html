Singular Control

rem: Singular Control

Find \(u\) over \(t \in [0,5]\) to minimize \[ J = x_4(t_f) \] subject to: \[ \dot{x}_1 = x_2 \\ \dot{x}_2 = x_3 \\ \dot{x}_3 = u \\ \dot{x}_4 = x_1^2 \] The initial conditions are: \[ x(0) = [1 \ 0 \ 0 \ 0] \\ -1 \leq u \leq 1 \]

Reference:

  1. Rein Luus, Iterative Dynamic Programming, CHAPMAN & HALL/CRC Monographs and Surveys in Pure and Applied Mathematics.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010 

par:
  tf = 5

dyn:
  x1 x2 x3 x4	# state variables
  u	# control variable

lim:
  -1 <= u <= 1

t=t0:
  x1 = 1
  x2 = 0
  x3 = 0
  x4 = 0

t=tf:

ini:
  x1 = 1
  x2 = 0
  x3 = 0
  x4 = 0
  u = 0

exp:

equ:
  x1´ == x2
  x2´ == x3
  x3´ == u
  x4´ == sqr(x1)

obj:
  minimize final(x4) using nlp
