An optimal control problem

rem: An optimal control problem

Consider the scalar system \(\dot{x}=x+u\), given the constraint \(u(t+T)=u(t)\). Find the control \(u\) that drives \(x(0)=1\) to \(x(2T)=0\) and minimizes

\[ \int_0^{2T} u^2(t) \ \mathrm{d}t \]

An approach by Fourier series.

Reference:

  1. https://math.stackexchange.com/questions/3342856/ 

par:
  tf = 2

dyn:
  x
  u
  J

t=t0:
  x = 1
  J = 0

t=tf:
  x = 0

set:
  coef0 = c0:c128
  coef(coef0) = c1:c128

var:
  a[coef0]
  b[coef]

equ:
  x´ == x + u
  J´ == u*u
  u == a['c0'] + sum(coef, a[coef]*cos(ord(coef)*4*pi*Time/tf) + b[coef]*sin(ord(coef)*4*pi*Time/tf))

obj:
  minimize final(J) using nlp
