Aly-Chan problem

rem: Aly-Chan problem

This problem is taken from the examples implemented in ICLOCS2 (www.ee.ic.ac.uk/ICLOCS/).

References:

  1. G.M. Aly and W.C. Chan, Application of a modified quasilinearization technique to totally singular optimal problems, International Journal of Control, 17(4):809-815, April 1973.
  2. P. Falugi, E. Kerrigan, E. van Wyk, Imperial College London Optimal Control Software User Guide (ICLOCS), http://www.ee.ic.ac.uk/ICLOCS/user_guide.pdf 

par:
  tf = pi/2

dyn:
  x{1:3}
  u

lim:
  -1 <= u <= 1

t=t0:
  x{1:3} = {0 1 0}

equ:
  x1´ == x2
  x2´ == u
  x3´ == 0.5*(x2*x2-x1*x1)
  u :: spline3

obj:
  minimize final(x3) using nlp
