Drug Displacement Problem

rem: Drug Displacement Problem

The desired level of two drugs, warfarin and phenylbutazone, must be reached in a patients bloodstream in minimum time.

Find \(u\) over \(t \in [0,t_f]\) to minimize \[ J = t_f \] subject to: \[ \dot{x}_1 = g_1 \ (g_4 \ (0.02-x_1)+46.4 \ x_1 \ (u-2 x_2)) \\ \dot{x}_2 = g_1 \ (g_3 \ (u-2 x_2)+46.4 \ (0.02-x_1)) \\ g_2 = 1+0.2 \ (x_1+x_2) \\ g_3 = g_2^2+232+46.4 \ x_2 \\ g_4 = g_2^2+232+46.4 \ x_1 \\ g_1 = \frac{g_2^2}{g_3 \ g_4-2152.96 \ x_1 \ x_2} \\ 0 \leq u \leq 8 \] \(x_1\) is the concentration of warfarin, and \(x_2\) of phenylbutazone. The initial and final conditions are: \[ x_0 = [0.02 \ \ 0] \\ x_{t_f} = [0.02 \ \ 2.00] \]

Reference:

  1. Rein Luus, Iterative Dynamic Programming, CHAPMAN & HALL/CRC Monographs and Surveys in Pure and Applied Mathematics.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010 

var:
  tf

dyn:
  x1 x2	# state variables
  u	# control variable

lim:
  1 <= tf <= 500
  0 <= u <= 8

t=t0:
  x1 = 0.02
  x2 = 0

t=tf:
  x1 = 0.02
  x2 = 2

ini:
  tf = 300
  x1 = 0.02
  x2 = linspace(0,2)
  u = linspace(8,0)

exp:
  g2 == 1+0.2*(x1+x2)
  g3 == sqr(g2)+232+46.4*x2
  g4 == sqr(g2)+232+46.4*x1
  g1 == sqr(g2)/(g3*g4-2152.96*x1*x2)

equ:
  x1´ == g1*(g4*(0.02-x1)+46.4*x1*(u-2*x2))
  x2´ == g1*(g3*(u-2*x2)+46.4*(0.02-x1))

obj:
  minimize tf using nlp
