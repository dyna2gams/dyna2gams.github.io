Singular Continuous Stirred Tank Reactor

rem: Singular Continuous Stirred Tank Reactor

Nonlinear two-stage CSTR problem.

Find \(u\) over \(t \in [0,t_f]\) to minimize: \[ J = x(t_f)' x(t_f) + t_f \] (the state variables are moved to bounds) subject to: \[ \dot{x}_1 = -3 x_1+g_1 \\ \dot{x}_2 = -11.1558 x_2+g_1-8.1558 (x_2+0.1592) u_1 \\ \dot{x}_3 = 1.5 (0.5 x_1-x_3)+g_2 \\ \dot{x}_4 = 0.75 x_2-4.9385 x_4+g_2-3.4385 (x_4+0.122) u_2 \\ g_1 = 1.5e7 (0.5251-x_1) \exp(-\frac{10}{x_2+0.6932})-1.5e10 (0.4748+x_1) \exp(-\frac{15}{x_2+0.6932}) - 1.4280 \\ g_2 = 1.5e7 (0.4236-x_2) \exp(-\frac{10}{x_4+0.6560})-1.5e10 (0.5764+x_3) \exp(-\frac{15}{x4+0.6560}) - 0.5086 \] The initial conditions are: \[ x(0) = [0.1962 \ \ -0.0372 \ \ 0.0946 \ \ 0] \\ -1 \leq u_{1:2} \leq 1 \]

Reference:

  1. R. Luus, Iterative Dynamic Programming, CHAPMAN & HALL/CRC Monographs and Surveys in Pure and Applied Mathematics.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010.

exp:

  g2 == 1.5e7*(0.4236-x2)*exp(-10/(x4+0.6560)) - 1.5e10*(0.5764+x3)*exp(-15/(x4+0.6560)) - 0.5086

equ:
  x1´ == -3*x1+g1
  x2´ == -11.1558*x2+g1-8.1558*(x2+0.1592)*u1
  x3´ == 1.5*(0.5*x1-x3)+g2
  x4´ == 0.75*x2-4.9385*x4+g2-3.4385*(x4+0.122)*u2
