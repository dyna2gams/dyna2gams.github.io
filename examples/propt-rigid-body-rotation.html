Rigid Body Rotation

rem: Rigid Body Rotation

Find \(u\) over \(t \in [0,1]\) to minimize: \[ J = \frac{1}{4} \int_0^{1} (u_1^2+u_2^2)^2 \ \mathrm{d}t \] subject to: \[ \dot{x} = a \ y+u_1 \\ \dot{y} = -a \ x+u_2 \\ \dot{u}_1 = a \ u_2 \\ \dot{u}_2 = -a \ u_1 \\ x(t_0) = [0.9 \ 0.75] \\ x(t_f) = [0 \ 0] \\ a = 2 \]

Reference:

  1. I. Ioslovich and P.-O. Gutman, On smooth optimal control determination, Automatica, 40(12):2175-2180, 2004.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010. 

par:
  a = 2
  tf = 1

dyn:
  x y u1 u2	# state variables
  J

t=t0:
  x = 0.9
  y = 0.75
  J = 0
  
t=tf:
  x = 0
  y = 0

equ:
  x´ == a*y+u1
  y´ == -a*x+u2
  u1´ == a*u2
  u2´ == -a*u1
  J´ == 0.25*sqr(sqr(u1)+sqr(u2))

obj:
  minimize final(J) using nlp
