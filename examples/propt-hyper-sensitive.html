Hyper Sensitive Optimal Control

rem: Hyper Sensitive Optimal Control

Find \(u(t)\) over \(t \in [0,t_f]\) to minimize \[ J = \int_0^{t_f} (x^2 + u^2) \ \mathrm{d}t \] subject to: \[ \dot{x} = -x^3+u \\ x_0 = 1 \\ x_{t_f} = 1.5 \\ t_f = 10 \]

Reference:

  1. Anil V. Rao and Kenneth D. Mease, Eigenvector approximate dichotomic basis method for solving hyper-sensitive optimal control problems, Optimal Control Applications and Methods, Volume 21, Issue 1, January/February 2000, Pages 1-19.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010 

par:
  tf = 10

dyn:
  x	# state variables
  u	# control variable
  J

t=t0:
  x = 1
  J = 0
  
t=tf:
  x = 1.5

ini:
  x = 0
  u = 0

equ:
  x´ == -x*x*x + u
  J´ == sqr(x) + sqr(u)

obj:
  minimize final(J) using nlp
