Two-Link Robot

rem: Two-Link Robot

Singular time-optimal 2 Link robot control

Reference:

  1. L.G. Van Willigenburg, Computation of time-optimal controls applied to rigid manipulators with friction, Int. J. Contr., Vol. 54, no 5, pp. 1097-1117 (1991).

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010 

par:
  mm11 = 5.775
  mm12 = 0.815
  mm22 = 0.815
  hm11 = 1.35
  m1 = 30.0
  m2 = 15

var:
  tf

dyn:
  x{1:4}	# state variables
  u{1:2}	# control variables

lim:
  0.75 <= tf <= 1.5
  -25 <= u1 <= 25
  -9 <= u2 <= 9

t=t0:
  x{1:4} = 0

t=tf:
  x{1:4} = {1.5 0 0 0}

ini:
  x{1:4} = 0
  u{1:2} = 0

exp:
  ms1 == mm11+2*hm11*cos(x2)
  ms2 == mm12+hm11*cos(x2)
  det == ms1*mm22-ms2*ms2
  ms11 == mm22/det
  ms12 == -ms2/det
  ms22 == ms1/det
  qg1 == -hm11*sin(x2)*(x4*x4+2*x3*x4)
  qg2 == hm11*sin(x2)*x3*x3

equ:
  x1´ == x3
  x2´ == x4
  x3´ == ms11*(u1-qg1)+ms12*(u2-qg2)
  x4´ == ms12*(u1-qg1)+ms22*(u2-qg2)

obj:
  minimize tf using nlp
