A nonlinear optimal control problem

rem: A nonlinear optimal control problem

Reference:

  1. M.H.N. Skandari, E. Tohidi, Numerical Solution of a Class of Nonlinear Optimal Control Problems Using Linearization and Discretization, Applied Mathematics, 2:646-652, 2011. 

par:
  tf = 1

dyn:
  x u J

lim:
  -1 <= u <= 1

t=t0:
  x = 0.9
  J = 0

t=tf:
  x = 0.4

ini:
  x = linspace(initial(x),final(x))
  u = uniform(-1,1)

equ:
  x´ == Time*(sqr(sqr(Time))-Time+1)*x - abs(u*u*u)*exp(sin(2*Pi*Time))
  J´ == (abs(sin(2*Pi*Time))-exp(-Time))*x

obj:
  minimize final(J) using dnlp

gms:
  @solveloop Method=movavg, Paraml=5 1, Varl=u
