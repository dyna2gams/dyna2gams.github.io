Flow in a Channel

rem: Flow in a Channel

Find \(u(t)\) over \(t \in [0,1]\) to minimize \[ J = 0 \] subject to: \[ \frac{d^{4}u}{dt^{4}} = R \ (\frac{du}{dt} \ \frac{d^{2}u}{dt^{2}}-u \ \frac{d^{3}u}{dt^{3}}) \\ u_0 = 0 \\ u_1 = 1 \\ \frac{du}{dt}(0) = 0 \\ \frac{du}{dt}(1) = 0 \\ R = 10 \] After some transformation we get this problem: \[ \dot{x}_1 = x_2 \\ \dot{x}_2 = x_3 \\ \dot{x}_3 = x_4 \\ \dot{x}_4 = R \ (x_2 \ x_3-x_1 \ x_4) \\ x_1(0) = 0 \\ x_1(1) = 1 \\ x_2(0) = 0 \\ x_2(1) = 0 \]

Reference:

  1. E.D. Dolan, J.J. Moré and T.S. Munson, Benchmarking Optimization Software with COPS 3.0, Argonne National Laboratory, Technical Report ANL/MCS-TM-273, February 2004.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010. 

par:
  R = 10
  tf = 1

dyn:
  x{1:4}	# state variables

lim:
  -inf <= x{1:4} <= +inf

t=t0:
  x1 = 0
  x2 = 0

t=tf:
  x1 = 1
  x2 = 0

ini:
  x1 = 3*Time^2 - 2*Time^3
  x2 = 2*Time - 6*Time^2
  x3 = Time - 12*Time
  x4 = -12

equ:
  x1´ == x2
  x2´ == x3
  x3´ == x4
  x4´ == R*(x2*x3-x1*x4)

obj:
  # Feasibility problem
  minimize 1 using nlp
