Two link arm problem

rem: Two link arm problem

Source: PSOPT - Optimal Control Solver, User Manual, Release 4 build 2015. 

var:
  tf

dyn:
  x{1:4}  # state variables
  u{1:2}  # control variables
  
lim:
  -2 <= x{1:4} <= 2
  -1 <= u{1:2} <= 1
  1 <= tf <= 10
  
t=t0:
  x1 = 0 
  x2 = 0
  x3 = 0.5
  x4 = 0
  
t=tf:
  x1 = 0 
  x2 = 0
  x3 = 0.5
  x4 = 0.522
  
ini:
  x1 = 0 
  x2 = 0
  x3 = 0.5
  x4 = 0.522
  tf = 3
  
exp:
  num1 == sin(x3)*((9/4)*cos(x3)*x1*x1 + 2*x2*x2) + (4/3)*(u1-u2) - (3/2)*cos(x3)*u2
  num2 == -( sin(x3)*((7/2)*x1*x1 + (9/4)*cos(x3)*x2*x2) - (7/3)*u2 + (3/2)*cos(x3)*(u1-u2) )
  den == 31/36 + 9/4*sqr(sin(x3))

equ:
  x1´ == num1/den
  x2´ == num2/den
  x3´ == x2 - x1
  x4´ == x1

obj:
  minimize tf using nlp
