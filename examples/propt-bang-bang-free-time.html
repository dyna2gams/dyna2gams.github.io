A Linear Problem with Bang Bang Control

rem: A Linear Problem with Bang Bang Control

This is a textbook bang-bang optimal control problem.

Find \(u\) over \(t \in [0,t_f]\) to minimize \[J = t_f\] subject to: \[ \dot{x}_1 = x_2 \\ \dot{x}_2 = u \\ x_1(0) = 0 \\ x_1(t_f) = 300 \\ x_2(0) = 0 \\ x_2(t_f) = 0 \\ -2 \leq u \leq 1 \]

Reference:

  1. J. Liang, Y.Q. Chen, M.Q.-H. Meng, R. Fullmer, Solving Tough Optimal Control Problems by Network Enabled Optimization Server (NEOS), Utah State University and Chinese University of Hong Kong, January 2003.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010. 

var:
  tf

dyn:
  x1 x2	# states
  u	# control

lim:
  10 <= tf <= 40
  -2 <= u <= 1

t=t0:
  x1 = 0
  x2 = 0

t=tf:
  x1 = 300
  x2 = 0

ini:
  tf = 20
  x1 = linspace(initial(x1),final(x1))
  x2 = linspace(initial(x2),final(x2))
  u = linspace(1,-1)

equ:
  x1´ == x2
  x2´ == u

obj:
  minimize tf using nlp
