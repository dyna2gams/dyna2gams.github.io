Bridge Crane System

rem: Bridge Crane System

Find \(u\) over \(t \in [0,t_f]\) to minimize \[ J = t_f \] subject to: \[ \dot{x}_1 = x_2 \\ \dot{x}_2 = u \\ \dot{x}_3 = x_4 \\ \dot{x}_4 = -0.98 \ x_3 + 0.1 \ u \] The initial conditions are: \[ x(0) = [0 \ 0 \ 0 \ 0] \\ x(t_f) = [15 \ 0 \ 0 \ 0] \\ -1 \leq u \leq 1 \]

Reference:

  1. Rein Luus, Iterative Dynamic Programming, CHAPMAN & HALL/CRC Monographs and Surveys in Pure and Applied Mathematics.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010 

var:
  tf

dyn:
  x1 x2 x3 x4	# state variables
  u	# control variable

lim:
  0.1 <= tf <= 100
  -1 <= u <= 1

t=t0:
  x1 = 0
  x2 = 0
  x3 = 0
  x4 = 0

t=tf:
  x1 = 15
  x2 = 0
  x3 = 0
  x4 = 0

ini:
  u = linspace(1,-1)

equ:
  x1´ == x2
  x2´ == u
  x3´ == x4
  x4´ == -0.98*x3+0.1*u

obj:
  minimize tf using nlp
