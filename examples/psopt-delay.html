Time delay problem

rem: Time delay problem

Source: PSOPT - Optimal Control Solver, User Manual, Release 4 build 2015 

par:
  tf = 5
  
dyn:
  x{1:3}   # state variables
  u        # control variable
  z{1:2}   # auxiliary variables	
  
lim:
  -100 <= x{1:3} <= 100
  -100 <= u <= 100
    
t=t0:
  x1 = 1
  x2 = 1
  x3 = 0
  
ini:
  x1 = 1
  x2 = 1
  x3 = 0
  u = 0

equ:
  x1´ == x2
  x2´ == -10*x1-5*x2-2*z1-z2+u
  x3´ == 0.5*(10*x1*x1+x2*x2+u*u)
  z1  == delay(x1,0.25)
  z2  == delay(x2,0.25)

obj:
  minimize final(x3) using nlp
