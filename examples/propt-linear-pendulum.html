Linear Pendulum

rem: Linear Pendulum

A linear pendulum problem controlled by an external force.

Find \(u\) over \(t \in [0,20]\) to maximize: \[ J = x_1(t_f) \] subject to: \[ \dot{x}_1 = x_2 \\ \dot{x}_2 = u-x_1 \\ x(t_0) = [0 \ 0] \\ |u| \leq 1 \]

Reference:

  1. A. Bressan, Viscosity Solutions of Hamilton-Jacobi Equations and Optimal Control Problems, S.I.S.S.A, Trieste, Italy, spring 2011.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010. 

par:
  tf = 20

dyn:
  x1 x2	# state variables
  u	# control variable

lim:
  -1 <= u <= 1

t=t0:
  x1 = 0
  x2 = 0

ini:
  x1 = 0
  x2 = 0
  u = 0

equ:
  x1´ == x2
  x2´ == u - x1

obj:
  maximize final(x1) using lp
