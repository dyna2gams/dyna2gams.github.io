Discrete simulation of a continuous transfer function

rem: Discrete simulation of a continuous transfer function

\[ y = \dfrac{K(1-s \ T_0)}{(1+s \ T_1)^2} u \\ K = 1; \ T_1 = 8; \ T_0 = 4 \\ T_s = 1; \ \delta = 0.5 \]

where \(T_s\) is the sampling time and \(\delta\), the delay. 

run:
  time-model = Discrete
  N = 50
  
par:
  ts = 1
    
dyn:
  x{1:2}
  y u
  
k=k0:
  x{1:2} = 0

sim:
  u := TimeStep(5*ts)
  x1´ := 0.882 * x1 + 0.118 * u
  x2´ := 0.1655 * x1 + 0.882 * x2 - 0.048 * u
  y := 0.0880 * x1 + 0.939 * x2 - 0.0275 * u
