Dielectrophoresis Particle Control

rem: Dielectrophoresis Particle Control

Find \(u\) over \(t \in [0,t_f]\) to minimize: \[ J = t_f \] subject to: \[ \dot{x} = y \ u+\alpha \ u^2 \\ \dot{y} = -c \ y+u \\ |u| \leq 1 \\ \alpha = -\frac{3}{4} \\ c = 1 \\ [x_0 \ y_0] = [1 \ 0] \\ x_{t_f} = 2 \]

Reference:

  1. D.E. Chang, N. Petit and P. Rouchon, Time-Optimal Control of a Particle in a Dielectrophoretic System, IEEE Transactions on Automatic Control, 51(7):1100-1114, 2006.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010. 

var:
  tf

dyn:
  x y	# state variables
  u	# control variable

lim:
  sqrt(eps) <= x <= +inf
  sqrt(eps) <= y <= +inf
  1 <= tf <= 100
  -1 <= u <= 1

t=t0:
  x = 1
  y = 0

t=tf:
  x = 2

ini:
  tf = 10
  x = linspace(1,2)
  y = linspace(0,1)
  u = 1

equ:
  x´ == y*u-3/4*sqr(u)
  y´ == -y+u

obj:
  minimize tf using nlp
