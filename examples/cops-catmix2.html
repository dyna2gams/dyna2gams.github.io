Catalyst Mixing Problem (COPS Problem 14)

rem: Catalyst Mixing Problem (COPS Problem 14)

The Catalyst mixing problem seeks an optimal policy for mixing two catalysts "along the length of a tubular plug of reactor involving several reactions".

See: COPS: Large-Scale Optimization Problems 

set: 
  ne = e1:e2   # Number of differential equations

par:
  tf = 1 
  bc(ne) = [e1 1, e2 0]   # Boundary conditions for x

dyn: v[ne] u

lim: 0 <= u <= 1

t=t0:
  v[ne] = bc(ne)

equ:
  v´['e1'] == u*(10*v['e2']-v['e1'])
  v´['e2'] == u*(v['e1'] - 10*v['e2']) - (1 - u)*v['e2']

obj: minimize -1 + sum(ne, final(v[ne])) using nlp
