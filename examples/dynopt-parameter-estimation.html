Parameter Estimation Problem

rem: Parameter Estimation Problem

Find \(p_1\) and \(p_2\) over \(t \in [0,6]\) to minimize \[ J = \sum_{i=1,2,3,5} ( x_1(t_i) - x_1^{m}(t_i) )^2 \] subject to: \[ \dot{x}_1 = x_2 \\ \dot{x}_2 = 1-2 x_2-x_1 \] where \[ x(0) = [p_1 \ p_2] \\ t_i = [1 \ 2 \ 3 \ 5] \\ x_1^{m}(t_i) = [0.264 \ 0.594 \ 0.801 \ 0.959] \\ -1.5 \leq p_{1:2} \leq 1.5 \]

Source: DYNOPT - MATLAB Dynamic Optimisation Code, User’s Guide version 4.3.0, March 28, 2017 

par:
  tf = 6

dyn:
  x1 x2	# state variables

obs:
  Time     x1
  1     0.264 
  2     0.594 
  3     0.801 
  5     0.959

t=t0:
  -1.5 <= x1 <= 1.5
  -1.5 <= x2 <= 1.5

ini:
  x1 = ObservedVal(x1)

equ:
  x1´ == x2
  x2´ == 1-2*x2-x1

obj:
  minimize ObservedErr(x1) using nlp
