Second Order System

rem: Second Order System

Optimal control of a second order system.

Find \(u\) over \(t \in [0,2]\) to minimize \[ J = \frac{1}{2} \int_0^{2} u^2 \ \mathrm{d}t \] subject to: \[ \dot{x}_1 = x_2 \\ \dot{x}_2 = u \\ x_1(0) = 1 \\ x_1(2) = 0 \\ x_2(0) = 1 \\ x_2(2) = 0 \\ -100 \leq u \leq 100 \]

Reference:

  1. M. Čižniar, M. Fikar, M.A. Latifi, MATLAB™ Dynamic Optimisation Code DYNOPT User’s Guide, Technical Report, KIRP FCHPT STU Bratislava, Slovak Republic, 2006.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010. 

par:
  tf = 2

dyn:
  x1 x2	# state variables
  u	# control variable
  J
  
lim:
  -100 <= x1 <= 100
  -100 <= x2 <= 100
  -100 <= u <= 100

t=t0:
  x1 = 1
  x2 = 1
  J = 0

t=tf:
  x1 = 0
  x2 = 0

ini:
  x1 = linspace(1,0)
  x2 = linspace(-1,0)
  u = linspace(-3.5,2.5)

equ:
  x1´ == x2
  x2´ == u
  J´ == sqr(u)/2

obj:
  minimize final(J) using nlp
