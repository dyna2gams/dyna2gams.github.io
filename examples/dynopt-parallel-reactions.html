Parallel Reactions in Tubular Reactor

rem: Parallel Reactions in Tubular Reactor

Batch reactor with reactions: A -> B and A -> C.

Find \(T\) over \(t \in [0,1]\) to maximize \[ J = x_2(t_f) \] subject to: \[ \dot{x}_1 = -(u+0.5 u^2) \ x_1 \\ \dot{x}_2 = u \ x_1 \] where \[ x(0) = [1 \ 0] \\ 0 \leq u \leq 5 \]

Source: DYNOPT - MATLAB Dynamic Optimisation Code, User’s Guide version 4.3.0, March 28, 2017 

par:
  tf = 1

dyn:
  x1 x2	# state variables
  u	# control variable

lim:
  0 <= u <= 5

t=t0:
  x1 = 1
  x2 = 0

ini:
  x1 = 1
  x2 = 0
  u = linspace(0,5)

equ:
  x1´ == -(u+0.5*sqr(u))*x1
  x2´ == u*x1

obj:
  maximize final(x2) using nlp
