MPC control of a paper machine headbox

rem: MPC control of a paper machine headbox

This problem contemplates the control of consistency (percentage of pulp fibers in aqueous suspension) and liquid level in a paper machine headbox.

Version with state/disturbance observer in continuous time.

References:

  1. Maciejowski, J.M., Predictive control with constraints, Prentice Hall, 2000.
  2. Morari, M. and Ricker, N.L., Model Predictive Control Toolbox: User’s Guide, The Mathworks, 1995.
  3. Ying, Y., Rao, M., and Sun, Y., Bilinear Control Strategy For Paper-Making Process, Chemical Engineering Communications, 111(1), 13-28, 1992. 

par:
  te = 1
  tf = 30
  nlin = 1

mpc:
  sampling-time = te
  horizon = K0:K10
  states = x{1:4}
  inputs = u{1:2}
  disturbances = w{1:2}
  outputs = z{1:2}
  equations =
    z1 == x2
    z2 == x4
    x1´ == -1.930*x1 + 1.274*u1 + 1.274*u2
    x2´ ==  0.394*x1 - 0.426*x2
    x3´ == -0.630*x3 + 1.340*u1 - 0.650*u2 + 0.203*w1 + 0.406*w2 - 0.327*x3*u1$nlin - 0.327*x3*u2$nlin
    x4´ ==  0.820*x1 - 0.784*x2 + 0.413*x3 - 0.426*x4
  lb =
    u1 -10
    u2 -10
  ub =
    u1 +10
    u2 +10
  Q =    z1  z2
    z1   1   0
    z2   0   1
  horizon-Q = K1:K10
  R =    u1   u2
    u1   0.4  0
    u2   0    0.4
  horizon-R = K0:K2
  decay = exp(-te)	# Tref = 1; decay = exp(-te/Tref)

dyn:
  x{1:4}	# feed tank level / headbox level / feed tank consistency / headbox consistency
  y{1:3}	# measured outputs: headbox level / feed tank consistency / headbox consistency
  z{1:2}	# controlled variables
  u{1:2}	# stock flow rate / white water flow rate
  w{1:2}	# disturbances (measured, unmeasured)
  oz{1:2}	# state variables of the observer
  ox1		# estimation of x1
  ow2		# estimation of w2

t=t0:
  x{1:4} = 0
  oz{1:2} = 0

par:					# pseudo parameters updated by dcc
  tm = 0
  tk = 0
  u{1:2}e

mac:
  @dcc(u1,u2)				# digital computer control
    if(Time >= tf/20,			# desired output: no change to the setpoint
      mpc_s('Z{1:2}',kk_) = 0;
    else
      mpc_s('Z{1:2}',kk_) = 0;
      );
    if(Time >= tm,			# compare current time and sampling time
					# sampling
      mpc_uo('U{1:2}') = last(u{1:2});		# ... of the control variables
      mpc_x.fx('X{2:4}','K0') = last(y{1:3});	# of the measured state variables
      mpc_w('W1',kk_) = last(w1);		# ... of the measured disturbance
      mpc_x.fx('X1','K0') = last(ox1);		# ... of the observer estimations
      mpc_w('W2',kk_) = last(ow2);
      tm = tm + te			# reset sampling time
      );
    if(Time >= tk,			# compare current time and control time
      solve mpc minimizing mpc_ksi using nlp;
      &u{1:2} = mpc_u.l('U{1:2}','K0');	# update control with outcome of the MPC (&u{1:2} are the arguments of dcc)
      tk = tk + te			# reset control time
      );

ini:
  w1 = ifthen(Time>tf/20,1,0)		# define the disturbances
  w2 = 0

sim:
  # plant model
  y1 := x2
  y2 := x3
  y3 := x4
  z1 := x2
  z2 := x4
  @dcc(u1e,u2e)
  u1 := u1e
  u2 := u2e
  x1´ := -1.930*x1                                  + 1.274*u1 + 1.274*u2
  x2´ :=  0.394*x1 - 0.426*x2
  x3´ :=                      - 0.630*x3            + 1.340*u1 - 0.650*u2 - 0.327*x3*u1$nlin - 0.327*x3*u2$nlin
  x4´ :=  0.820*x1 - 0.784*x2 + 0.413*x3 - 0.426*x4
  # observer model
  oz1´ := -0.5*oz1 - 1.367*y1 + 0.720*y2 + 0.129*y3 + 1.274*u1 + 1.274*u2
  oz2´ := -0.2*oz2 + 0.211*y2 - 0.0103*w1 + 0.659*u1 + 0.315*u2 + 0.161*y2*u1$nlin + 0.161*y2*u2$nlin
  ox1 := oz1 - 1.744*y3
  ow2 := oz2 + 0.493*y2 + 0.226*w1
