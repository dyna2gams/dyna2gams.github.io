The Prothero-Robinson stiff ODE model

rem: The Prothero-Robinson stiff ODE model


par:
  epsilon = 0.1
  omega = 100
  gamma = -1e+4
  tf = 3.5

dyn:
  y{1:2}
  
t=t0:
  y{1:2} = 0
  
sim:
  y1´ :=   gamma*(y1-sin(Time)) + epsilon*(y2-sin(omega*Time)) + cos(Time)
  y2´ := epsilon*(y1-sin(Time)) -         (y2-sin(omega*Time)) + omega*cos(omega*Time)
