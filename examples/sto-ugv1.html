UGV stochastic path planning problem

rem: UGV stochastic path planning problem

We consider the dynamics of a simple two-wheeled unmanned ground vehicle (UGV) with differential drive. We would like to control the velocity \(u_1(t)\) and the steering angle \(u_2(t)\) of the UGV in a way that maximizes the probability that the UGV hits a target located at \((x_1,x_2) = (3,3)\) at \(t_f = 10\) under uncertain initial conditions and wheel radius \(R\).

Reference:

  1. P. Lambrianides, Q. Gong, D. Venturi, A new scalable algorithm for computational optimal control under uncertainty, arXiv:1909.07960v1 [math.NA] 17 Sep 2019. 

set:
  scen = sce1:sce64

par:
  R[scen] = uniform(1,1.5)
  aR = sum(scen, R[scen])/card(scen)
  q = 0.01
  tf = 10

dyn:
  x{1:3}[scen]
  u{1:2}
  J
  ax{1:3}
  au{1:2}
  aJ

lim:
  -1 <= u{1:2} <= 1
  -1 <= au{1:2} <= 1

t=t0:
  x{1:3}[scen] = uniform(-0.05,0.05)
  ax{1:3} = sum(scen, x{1:3}[scen])/card(scen)
  J = 0
  aJ = 0

equ:
  x1´[scen] == R[scen]*u1*cos(x3[scen])
  x2´[scen] == R[scen]*u1*sin(x3[scen])
  x3´[scen] == R[scen]*u2
  J´ == sqr(u1) + sqr(u2)
  ax1´ == aR*au1*cos(ax3)
  ax2´ == aR*au1*sin(ax3)
  ax3´ == aR*au2
  aJ´ == sqr(au1) + sqr(au2)

obj:
  minimize \
    0.5*sum(scen,sqr(final(x1[scen])-3)+sqr(final(x2[scen])-3))/card(scen)+0.5*q*final(J) + \
    0.5*(sqr(final(ax1)-3)+sqr(final(ax2)-3))+0.5*q*final(aJ) \
    using nlp

gms:
  scalars x1a, x2a;
  x1a = sum(scen, final(x1.l[scen]))/card(scen);
  x2a = sum(scen, final(x2.l[scen]))/card(scen);
  display x1a, x2a;
