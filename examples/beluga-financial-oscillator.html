Financial oscillator

rem: Financial oscillator

Reference:

  1. P. Chen and S.M.N. Islam, Optimal Control Models in Finance - A New Computational Approach, Springer-Verlag New York Inc., 2005.

See: Beluga 

par:
  T = 5
  B = 0.2
  x10 = 3
  x20 = 5
  epsilon = 1e-6
  tf = 1

dyn:
  x1
  x2
  u
  J

lim:
  -2 <= u <= 2

t=t0:
  J = 0
  x1 = x10
  x2 = x20

equ:
  x1´ == T*x2
  x2´ == -T*x1 + T*u - T^2*B*x2
  #J´ == sqrt(epsilon+sqr(x1 + 5*Time - 5))
  J´ == sqr(x1 + 5*Time - 5)

ini:
  x1 = initial(x1)
  x2 = initial(x2)

obj:
  minimize final(J) using nlp
