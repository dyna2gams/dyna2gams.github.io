Optimal fishing plan

rem: Optimal fishing plan

A population of \(P\) fish in a certain lake grow at rate

\[ \dot{P}(t) = a P(t) - b P^2(t) \]

if undisturbed by people. Fish can be withdrawn from the lake and consumed at rate \(c(t)\), yielding utility \(u(c(t))\) to the consuming community and reducing the fish growth rate accordingly.

\[ \dot{P}(t) = a P(t) - b P^2(t) - c(t) \]

Assume future utilities to the community are discounted at constant rate \(r\). Characterize a fishing (consumption) plan to maximize the present value of the discounted stream of utilities.

\[ \max J = \int_0^\infty w\big(c(t)\big)e^{-rt} \ \mathrm{d}t \]

Assume that \(P(0)=a/b\), and that \(u'>0\), \(u''<0\).

Reference:

  1. https://math.stackexchange.com/questions/3398821/ 

par:
  r = 0.05
  beta = 0.3
  a = 5
  b = 0.05
  tf = 100

dyn:
  P	# fish population
  c	# fish consumption
  J	# criterion

lim:
  0 <= P <= +inf
  0 <= c <= +inf

t=t0:
  P = a/b
  J = 0

equ:
  P´ == a*P - b*P*P - c
  J´ == c^beta*exp(-r*Time)

obj:
  maximize final(J) using nlp
