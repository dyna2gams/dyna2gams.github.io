Nishida problem

rem: Nishida problem

This case study was presented by Nishida et al. (1976) and it is posed as follows:

Minimize: \[ J = x_1(t_f)^2+x_2(t_f)^2+x_3(t_f)^2+x_4(t_f)^2 \] subject to: \[ \dot{x}_1 = -0.5 \ x_1+5 \ x_2 \\ \dot{x}_2 = -5 \ x_1-0.5 \ x_2+u \\ \dot{x}_3 = -0.6 \ x_3+10 \ x_4 \\ \dot{x}_4 = -10 \ x_3-0.6 \ x_4+u \\ -1 \leq u \leq 1 \] with the initial conditions: \(x_i(0) = 10, i=1,\dotso,4\) and with \(t_f = 4.2\).

Reference:

  1. Vassilios S. Vassiliadis, Eva Balsa Canto, Julio R. Banga, Second-order sensitivities of general dynamic systems with application to optimal control problems, Chemical Engineering Science, Volume 54, Issue 17, September 1999, Pages 3851-3860.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010 

par:
  tf = 4.2

dyn:
  x{1:4}	# state variables
  u	# control variable

lim:
  -15 <= x{1:4} <= 15
  -1 <= u <= 1

t=t0:
  x{1:4} = 10

ini:
  x{1:4} = linspace(10,0)

equ:
  x1´ == -0.5*x1+5*x2
  x2´ == -5*x1-0.5*x2+u
  x3´ == -0.6*x3+10*x4
  x4´ == -10*x3-0.6*x4+u

obj:
  minimize sqr(final(x1))+sqr(final(x2))+sqr(final(x3))+sqr(final(x4)) using nlp
