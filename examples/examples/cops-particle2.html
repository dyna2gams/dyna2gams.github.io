rem: Particle Steering (COPS Problem 9)

See: COPS: Large-Scale Optimization Problems 

par:
  a = 100

set:
  i = 1:4

var:
  tf

dyn:
  y[i] u

lim:
  0 <= tf <= 10
  -pi/2 <= u <= +pi/2

t=t0:
  y[i] = 0

t=tf:
  y[2] = 45
  y[3] = 5
  y[4] = 0

ini:
  tf = 1
  y[i] = 0

equ:
  y´[1] == y[2]
  y´[2] == a*cos(u)
  y´[3] == y[4]
  y´[4] == a*sin(u)

obj:
  minimize tf using nlp
