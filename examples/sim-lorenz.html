The Lorenz model

rem: The Lorenz model

We look at the Lorenz system of equations:

\[ \dot{x} = \sigma(y - x) \\ \dot{y} = x (\rho - z) - y \\ \dot{z} = x y - \beta z \]

The parameter \(\rho\) is the ratio of the Rayleigh number divided by the critical Rayleigh number. The parameter \(\sigma\) is the Prandtl number. The third parameter \(\beta\) is related to the horizontal wave number of the system. Typical values are \(\rho = 28\), \(\sigma = 10\), \(\beta = 8/3\). A common second set of values is \(\rho = 45.92\), \(\sigma = 16\), \(\beta = 4\). 

run:
  iterations-number = 8

par:
  rho = 28
  sigma = 10
  beta = 8/3
  tf = 100

dyn:
  x y z
  
t=t0:
  x = 1
  y = 1
  z = 1
  
sim:
  x´ := sigma*(y-x)
  y´ := x*(rho-z) - y
  z´ := x*y - beta*z

gpl:
  @plotZXY z, y, x, "@TITLE@"
