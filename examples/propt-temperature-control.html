Temperature Control

rem: Temperature Control

Heating a room using the least possible energy.

Find \(u\) over \(t \in [0,1]\) to minimize: \[ J = \frac{1}{2} \int_0^{1} u^2 \ \mathrm{d}t \] subject to: \[ \dot{x} = -2 x + u \\ x(0) = 0 \\ x(1) = 10 \]

Reference:

  1. Victor M. Becerra, Optimal Control CY3H2, Lecture notes, School of Systems Engineering, University of Reading.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010 

par:
  tf = 1

dyn:
  x	# state variables
  u	# control variable
  J
  
lim:
  0 <= u <= +inf

t=t0:
  x = 0
  J = 0

t=tf:
  x = 10

ini:
  x = linspace(0,10)
  u = 1

equ:
  x´ == -2*x+u
  J´ == 0.5*sqr(u)

obj:
  minimize final(J) using nlp
