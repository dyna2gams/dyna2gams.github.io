Stirred Tank

rem: Stirred Tank

Find \(u\) over \(t \in [0,0.78]\) to minimize

(5a) \[ J = \frac{1}{2} \int_0^{t_f} (x_1^2+x_2^2+0.1 u^2) \ \mathrm{d}t + x_1(t_f)^2 + x_2(t_f)^2 \] (5b) \[ J = \frac{1}{2} \int_0^{t_f} (x_1^2+x_2^2+0.1 u^2) \ \mathrm{d}t \] (5c) \[ J = \frac{1}{2} \int_0^{t_f} (x_1^2+x_2^2) \ \mathrm{d}t \] subject to: \[ a_1 = x_1 + 0.25 \\ a_2 = x_2 + 0.5 \\ a_3 = x_1 + 2 \\ a_4 = a_2 \exp(25 \frac{x_1}{a_3}) \\ \dot{x}_1 = -2 a_1 + a_4 - a_1 u \\ \dot{x}_2 = 0.5 - x_2 - a_4 \\ x(0) = [0.05 \ 0] \] (5b), (5c) \[x(t_f) = [0 \ 0]\] (5c) \[u \leq 1\]

References:

  1. Oskar von Stryk, User’s Guide for DIRCOL: A Direct Collocation Method for the Numerical Solution of Optimal Control Problems, Technische Universität Darmstad (1999).
  2. Kirk, D. E., Stirred-Tank Chemical Reactor - Optimal control theory: An introduction, Prentice-Hall, 1970.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010 

par:
  tf = 0.78

dyn:
  x1 x2	# state variables
  u	# control variable
  J

lim:
  -1.99 <= x1 <= 100
  -100 <= x2 <= 100
  -1000 <= u <= 1000

t=t0:
  x1 = 0.05
  x2 = 0
  J = 0

ini:
  x1 = 0.05
  x2 = 0
  u = 0

exp:
  a1 == x1 + 0.25
  a2 == x2 + 0.5
  a3 == x1 + 2.0
  a4 == a2*exp(25.0*x1/a3)

equ:
  x1´ == -2.0*a1 + a4 - a1*u
  x2´ == 0.5 - x2 - a4
  J´  == (sqr(x1) + sqr(x2) + 0.1*sqr(u))/2

obj:
  minimize final(J+sqr(x1)+sqr(x2)) using nlp
