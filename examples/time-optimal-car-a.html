Fixed mesh formulation. 

var: tf

dyn: x v u

lim:
  5 <= tf <= 50
  -2 <= u <= 1

t=t0:
  x = 0
  v = 0

t=tf:
  x = 300
  v = 0

ini:
  tf = 20
  x = 0
  v = 0

equ:
  x´ == v
  v´ == u

obj:
  minimize tf using nlp
