Rayleigh Unconstrained

rem: Rayleigh Unconstrained

Find \(u\) over \(t \in [0,2.5]\) to minimize \[ J = \int_0^{2.5} (x_1^2 + u^2) \ \mathrm{d}t \] subject to: \[ \dot{x}_1 = x_2 \\ \dot{x}_2 = -x_1+(1.4-0.14 x_2^2) x_2+4 u \\ x_1(0) = -5 \\ x_2(0) = -5 \]

Reference:

  1. Jinsong Liang, Lecture Notes for ECE/MAE 7360, Robust and Optimal Control (part 2) Fall 2003, Nov. 20, 2003, Utah State University at Logan.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010 

par:
  tf = 2.5

dyn:
  x1 x2	# state variables
  u	# control variable
  J
  
lim:
  -100 <= x1 <= 100
  -100 <= x2 <= 100
  -100 <= u <= 100

t=t0:
  x1 = -5
  x2 = -5
  J = 0

ini:
  x1 = -5
  x2 = -5
  u = 0

equ:
  x1´ == x2
  x2´ == -x1+(1.4-0.14*sqr(x2))*x2+4*u
  J´ == sqr(x1)+sqr(u)
  
obj:
  minimize final(J) using nlp
