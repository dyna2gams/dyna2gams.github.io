Bryson-Denham Problem (Short version)

rem: Bryson-Denham Problem (Short version)

The Bryson-Denham Problem.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010. 

par:
  x1max = 1/9

var:
  tf

dyn:
  x1 x2	# state variables
  u
  J

lim:
  0.001 <= tf <= 50
  0 <= x1 <= x1max
  -10 <= x2 <= 10

t=t0:
  x1 = 0
  x2 = 1
  J = 0

t=tf:
  x1 = 0
  x2 = -1

ini:
  tf = 0.5

equ:
  x1´ == x2
  x2´ == u
  J´ == 0.5*sqr(u)

obj:
  minimize final(J) using nlp
