Brachistochrone problem

rem: Brachistochrone problem

Source: PSOPT - Optimal Control Solver, User Manual, Release 4 build 2015. 

dyn:
  x y v   # state variables
  theta   # control variable

var: 
  tf

lim:
  0 <= x <= 20
  0 <= y <= 20
  0 <= v <= 20
  0 <= theta <= 2*pi
  0 <= tf <= 10

t=t0:
  x = 0
  y = 0 
  v = 0
  
t=tf:
  x = 2
  y = 2
  
ini:
  x = linspace(0,1)
  y = linspace(0,1)
  v = linspace(0,1)
  theta = 1
  tf = 2

equ:
  x´ == v*sin(theta)
  y´ == v*cos(theta)
  v´ == 9.8*cos(theta)
  
obj: 
  minimize tf using nlp
