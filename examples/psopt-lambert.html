Lambert problem

rem: Lambert problem

Source: PSOPT Optimal Control Solver, User Manual, Release 4 build 2015 

par:
  G = 6.6720e-11	# Universal gravity constant [m^3/(kg s^2)]
  Me = 5.9742e+24	# Mass of earth [kg]
  mu = G*Me		# [m^3/sec^2]
  tf = 76*60		# [sec]

dyn:
  r{1:3}
  v{1:3}   # state variables
  
lim:
  -15945.34e+4 <= r1 <= 15945.34e+4
  -10249.47e+4 <= r2 <= 10249.47e+4
  0 <= r3 <= 0
  -15945.34e+4/tf <= v1 <= 15945.34e+4/tf
  -10249.47e+4/tf <= v2 <= 10249.47e+4/tf
  0 <= v3 <= 0
  
t=t0:
  r1 = 15945.34e+3
  r2 = 0
  r3 = 0
  
t=tf:
  r1 = 12214.83899e+3
  r2 = 10249.46731e+3
  r3 = 0
  
ini:
  r1 = linspace(initial(r1),final(r1))
  r2 = linspace(initial(r2),final(r2))
  r3 = linspace(initial(r3),final(r3))
  v{1:3} = r{1:3}/tf

exp:
  rr == sqrt(r1*r1+r2*r2+r3*r3)
  rr3 == rr*rr*rr
  
equ:
  r{1:3}´ == v{1:3}
  v{1:3}´ == -mu*r{1:3}/rr3
  
obj: 
  minimize 0 using nlp   # dummy objective
