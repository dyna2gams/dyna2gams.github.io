Maximum Radius Orbit Transfer

rem: Maximum Radius Orbit Transfer

Example on pages 66-69 from the book of Bryson and Ho.

Reference:

  1. A.E. Bryson and Y.-C. Ho, Applied Optimal Control: Optimization, Estimation, and Control, Hemisphere Publishing, New York, 1975.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010. 

par:
  tf = 1
  r0 = 1
  mu = 11
  th = 1.55
  m0 = 1
  rm0 = -0.25

dyn:
  x{1:3}	# state variables
  u		# control variable

t=t0:
  x1 = r0
  x2 = 0
  x3 = sqrt(mu/r0)

t=tf:
  x2 = 0

ini:
  x1 = r0
  x2 = 0
  x3 = sqrt(mu/r0)
  u = 0

equ:
  x1´ == x2
  x2´ == x3*x3/x1-mu/(x1*x1)+th*sin(u)/(m0+rm0*Time)
  x3´ == -x2*x3/x1+th*cos(u)/(m0+rm0*Time)
  0 == final(x3) - sqrt(mu/final(x1))

obj:
  maximize final(x1) using nlp
