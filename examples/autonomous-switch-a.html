A system with autonomous switches

rem: A system with autonomous switches

Reference:

  1. A. Bemporad and M. Morari, Control of systems integrating logic, dynamics, and constraints, Automatica 35(3):407-427, 1999. 

run:
  time-model = Discrete
  N = 6
  
par:
  q = 10
  ts = 1

dyn:
  x{1:2}
  u
  J

lim:
  -1 <= u <= +1

t=t0:
  x{1:2} = {-1 1}
  J = 0

t=tf:
  x{1:2} = 0

exp:
  alpha == ifthen(x1<0,-pi/3,pi/3)

equ:
  x1´ == 0.8*cos(alpha)*x1 - 0.8*sin(alpha)*x2
  x2´ == 0.8*sin(alpha)*x1 + 0.8*cos(alpha)*x2 + u
  J´ == J + q*sqr(u) + sqr(x1) + sqr(x2)

obj:
  minimize final(J) using dnlp
