Bryson-Denham Problem

rem: Bryson-Denham Problem

Standard formulation for Bryson-Denham.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010. 

dyn:
  x{1:3}	# state variables
  u		# control variables

var: 
  tf

lim:
  0 <= x1 <= 1/9
  -10 <= x2 <= 10
  -10 <= x3 <= 10
  -5000 <= u <= 5000
  0.001 <= tf <= 50

t=t0:
  x1 = 0
  x2 = 1
  x3 = 0 
  
t=tf:
  x1 = 0
  x2 = -1
  
ini:
  x1 = linspace(initial(x1),final(x1))
  x2 = linspace(initial(x2),final(x2))
  x3 = 0
  u = 0
  tf = 0.5
  
equ:
  x1´ == x2
  x2´ == u
  x3´ == u*u/2

obj: 
  minimize final(x3) using nlp
