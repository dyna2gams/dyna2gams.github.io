Singular Arc Problem

rem: Singular Arc Problem

Find \(u(t)\) over \(t \in [0,t_f]\) to minimize \[ J = t_f \] subject to: \[ \dot{x}_1 = u \\ \dot{x}_2 = cos(x_1) \\ \dot{x}_3 = sin(x_1) \\ x_2(t_f) = x_3(t_f) = 0 \\ |u| \leq 2 \\ x(0) = [\frac{\pi}{2} \ 4 \ 0] \]

Reference:

  1. L. S. Jennings, M. E. Fisher, K. L. Teo, C. J. Goh, MISER3 Optimal Control Software: Theory and User Manual Version 2.0, Department of Mathematics, The University of Western Australia, Nedlands, WA 6907, Australia (1997).

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010 

var:
  tf

dyn:
  x1 x2 x3	# state variables
  u	# control variable

lim:
  2 <= tf <= 1000
  -2 <= u <= 2

t=t0:
  x1 = pi/2
  x2 = 4
  x3 = 0

t=tf:
  x2 = 0
  x3 = 0

ini:
  tf = 20
  x1 = linspace(pi/2,pi)
  x2 = linspace(4,0)
  x3 = 0
  u = 0

exp:

equ:
  x1´ == u
  x2´ == cos(x1)
  x3´ == sin(x1)

obj:
  minimize tf using nlp
