Batch Reactor Problem

rem: Batch Reactor Problem

Batch reactor with reactions: \(A \rightarrow B \rightarrow C\).

Find \(T\) over \(t \in [0,1]\) to maximize \[ J = x_2(t_f) \] subject to: \[ \dot{x}_1 = -k_1 \ x_1^2 \\ \dot{x}_2 = k_1 \ x_1^2-k_2 \ x_2 \\ k_1 = 4000 \ \exp(-\frac{2500}{T}) \\ k_2 = 620000 \ \exp(-\frac{5000}{T}) \] where \[ x(0) = [1 \ 0] \\ 298 \leq T \leq 398 \]

Source: DYNOPT - MATLAB Dynamic Optimisation Code, User’s Guide version 4.3.0, March 28, 2017 

par:
  tf = 1

dyn:
  x1 x2	# state variables
  T	# control variables

lim:
  298 <= T <= 398

t=t0:
  x1 = 1
  x2 = 0

ini:
  x1 = 1
  x2 = 0
  T = linspace(398,298)

exp:
  k1 == 4000*exp(-2500/T)
  k2 == 620000*exp(-5000/T)

equ:
  x1´ == -k1*sqr(x1)
  x2´ == k1*sqr(x1)-k2*x2

obj:
  maximize final(x2) using nlp
