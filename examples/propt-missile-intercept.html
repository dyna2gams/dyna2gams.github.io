Missile Intercept

rem: Missile Intercept

Find scalar \(w\) over \(t \in [0,t_f]\) to minimize: \[ J = 0 \] subject to: \[ \dot{x}_1 = x_3 \\ \dot{x}_2 = x_4 \\ \dot{x}_3 = \cos(w) \\ \dot{x}_4 = \sin(w) \\ x(t_0) = [0 \ 0 \ 0 \ 0] \\ x_{1:2}(t_f) = [4+1.5 t_f \ \ 1] \\ 0 \leq w \leq \frac{\pi}{4} \]

Reference:

  1. E.G. Wiens, Optimal Control, Intercept Missile, Egwald Mathematics.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010. 

var:
  tf w

dyn:
  x{1:4}	# state variables

lim:
  1 <= tf <= 1e4
  0 <= w <= pi/4

t=t0:
  x{1:4} = 0

t=tf:
  x2 = 1

ini:
  tf = 10
  w = 0.2

equ:
  x1´ == x3
  x2´ == x4
  x3´ == cos(w)
  x4´ == sin(w)
  4 + 1.5*tf == final(x1)

obj:
  # stop as soon as problem is feasible
  minimize 0 using nlp
