Van der Pol Oscillator

rem: Van der Pol Oscillator

This case study has been studied by several authors, for example Morison, Gritsis, Vassiliadis and Tanartkit, and Biegler.

The dynamic optimization problem is to minimize: \[ J = x_3(t_f) \] subject to: \[ \dot{x}_1 = (1-x_2^2) x_1-x_2+u \\ \dot{x}_2 = x_1 \\ \dot{x}_3 = x_1^2+x_2^2+u^2 \\ -0.3 \leq u \leq 1.0 \\ x(t_0) = [0 \ 1 \ 0] \\ t_f = 5 \]

Reference:

  1. E. Balsa-Canto, J.R. Banga, A.A. Alonso, V.S. Vassiliadis, Restricted second order information for the solution of optimal control problems using control vector parameterization, Journal of Process Control, 12(2):243-255, February 2002.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010. 

par:
  tf = 5

dyn:
  x1 x2 x3	# state variables
  u		# control variables

lim:
  -10 <= x1 <= 10
  -10 <= x2 <= 10
  -10 <= x3 <= 10
  -0.3 <= u <= 1

t=t0:
  x1 = 0
  x2 = 1
  x3 = 0

ini:
  x1 = 0
  x2 = 1
  x3 = 0
  u = -0.01

equ:
  x1´ == (1-sqr(x2))*x1 - x2 + u
  x2´ == x1
  x3´ == sqr(x1) + sqr(x2) + sqr(u)

obj:
  minimize final(x3) using nlp
