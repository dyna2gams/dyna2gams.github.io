Isoperimetric Constraint Problem

rem: Isoperimetric Constraint Problem

Find \(u\) over \(t \in [0,1]\) to minimize \[ J = \int_0^{1} x^2 \ \mathrm{d}t \] subject to: \[ \dot{x} = -\sin(x)+u \\ \int_0^{1} u^2 \ \mathrm{d}t = 10 \] The initial conditions are: \[ x(0) = 1 \\ x(1) = 0 \]

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010. 

par:
  tf = 1

dyn:
  x	# state variables
  u	# control variable
  J K

lim:
  -10 <= x <= 10
  -4 <= u <= 4

t=t0:
  x = 1
  J = 0
  K = 0

t=tf:
  x = 0
  K = 10

ini:
  x = 0
  u = 0

equ:
  x´ == -sin(x) + u
  J´ == x
  K´ == sqr(u)

obj:
  minimize final(J) using nlp
