The bouncing ball problem

rem: The bouncing ball problem

Calculate and plot the path of a rubber ball for several bounces along a level floor, given its initial location and velocity. Assume that the only external force is gravity and that the acceleration due to gravity is 9.8 m/s². The bouncing characteristics of the ball follow the rule that 80% of the vertical velocity is lost when the ball bounces, but no loss in horizontal velocity occurs.

From "System simulation : Programming Style and Language" W. Kreutzer 

par:
  tf = 20
  g = -9.81	# the gravity acceleration
  a = 0.8	# the loss of vertical velocity

mac:
  @SimEventProc y = 0; vy = a*abs(vy)

dyn:
  x y		# co-ordinates of the ball
  vx vy		# the velocity components in the x and y directions

t=t0:
  vx = 1; vy = 0
  x = 0; y = 10

sim:
  vx´ := 0; x´ := vx
  vy´ := g; y´ := vy
  @SimEvent(y < 0)
