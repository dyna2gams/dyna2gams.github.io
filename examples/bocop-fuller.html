Fuller problem

rem: Fuller problem

Here is a very classical example of a chattering phenomenon:

\[\min \int_0^{t_f} x^2(t) \ \mathrm{d}t; \quad \ddot x(t) = u(t) \in [-1,1].\]

The solution is, for large enough \(t_f\), bang-bang (i.e., with values alternately +/-1), the switching times geometrically converging to a value \(\tau > 0\), and then the (trivial) singular arc \(x=0\) and \(u=0\). These switches are not easy to reproduce numerically.

Reference:

  1. A.T. Fuller, Study of an optimum non-linear control system, Journal of Electronics and Control, 15:63-71, 1963.

See: BOCOP - The optimal control solver. 

par:
  tf = 3.5

  -1 <= u <= 1

  x1 = 0
  x2 = 1

t=tf:
  x1 = 0
  x2 = 0

  x1 = 0
  x2 = linspace(1,0)
  u = -1/tf
  j = 0
  
equ:
  x1´ == x2
  x2´ == u
  j´  == x1*x1
  
obj:
  minimize final(j) using nlp
