E5 problem

rem: E5 problem

Another chemical reaction problem, called “E5” in the collection by Enright, Hull & Lindberg (1975).

Reference:

  1. W.H. Enright, T.E. Hull, B. Lindberg, Comparing Numerical Methods For Stiff Systems of ODEs, BIT Numerical Mathematics, 15(1):10-48, March 1975. 

par:
  tf = 1000

dyn:
  x{1:4}
  
lim:
  0 <= x{1:4} <= 2e-3
  
t=t0:
  x1 = 1.76e-3
  x2 = 0
  x3 = 0
  x4 = 0

exp:
  z1 == 7.89e-10 * x1
  z2 == 1.1e+7 * x1 * x3
  z3 == 1.13e+9 * x2 * x3
  z4 == 1.13e+3 * x4
  
equ:
  x1´ == - z1 - z2
  x2´ == z1 - z3
  x3´ == z1 - z3 - z2 + z4
  x4´ == z2 - z4

obj:
  minimize final(x4) using nlp	# dummy objective
