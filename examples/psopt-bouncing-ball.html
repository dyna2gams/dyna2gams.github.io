Bouncing ball

rem: Bouncing ball

Source: PSOPT - Optimal Control Solver, User Manual, Release 4 build 2015. 

par:
  g = 9.81
  a = 100
  k = 1000
  c = 5
  m = 1

var:
  tf

dyn:
  y{1:2}
  u
  J

lim:
  -10 <= y1 <= 20
  -10 <= y2 <= 10
  -100 <= u <= 100
  1 <= tf <= 10

t=t0:
  y1 = 10
  y2 = 0
  J = 0

t=tf:
  y1 = 5
  y2 = 0

ini:
  u = 0

equ:
  y1´ == y2
  y2´ == (u - g + 0.5*(1-tanh(a*y1))*(-k*y1-c*y2))/m
  J´ == sqr(u)

obj:
  minimize final(J) using nlp
