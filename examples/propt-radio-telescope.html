Radio telescope

rem: Radio telescope

Time-optimal positioning of a radio telescope with bounded control.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010. 

var:
  tf

dyn:
  x1 x2 x3	# state variables
  u	# control variable

lim:
  0 <= tf <= 5
  -1 <= u <= 1

t=t0:
  x{1:3} = {0 0 0}

t=tf:
  x{1:3} = {1 0 0}

ini:
  tf = 5
  x{1:3} = {0 0 0}
  u = 0

equ:
  x1´ == x2
  x2´ == -0.5*x2-0.1*x2/sqrt(x2*x2+1e-4)+x3
  x3´ == -2*x3+2*u

obj:
  minimize tf using nlp
