Catalyst mixing problem

rem: Catalyst mixing problem

Source: PSOPT Optimal Control Solver, User Manual, Release 4 build 2015 

par:
  tf = 1

dyn:
  x{1:2}  # state variables
  u       # control variable
  
lim:
  0.9 <= x1 <= 1.0
  0.0 <= x2 <= 0.1
  0 <= u <= 1
  
t=t0:
  x1 = 1
  x2 = 0
  
t=tf:
  0 <= x1 <= 0.95 
  
ini:
  x1 = linspace(1,0.915)
  x2 = linspace(0,0.05)
  u = linspace(1,0)

equ:
  x1´ == u*(10*x2-x1)
  x2´ == u*(x1-10*x2) - (1-u)*x2

obj:
  maximize 1-final(x1+x2) using nlp
