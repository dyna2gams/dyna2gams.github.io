Euler Buckling Problem

rem: Euler Buckling Problem

Over \(t \in [0,1] minimize\) \[ J = z \] subject to: \[ \dot{x}_1 = x_2 \\ \dot{x}_2 = \frac{-z \ x_1}{x_3^2} \\ \int_0^1 x_3(t) \ \mathrm{d}t = 1 \\ x_1(0) = 0 \\ x_1(1) = 0 \\ x_2(0) = 1 \\ x_3(0) >= 0.5 \\ x_3(t) >= 0.5 \]

Reference:

  1. L. S. Jennings, M. E. Fisher, K. L. Teo, C. J. Goh, MISER3 Optimal Control Software: Theory and User Manual Version 2.0, Department of Mathematics, The University of Western Australia, Nedlands, WA 6907, Australia (1997).

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010 

par:
  tf = 1

var:
  z

dyn:
  x1 x2 x3 x4	# state variables
  J

lim:
  -10 <= x1 <= 10
  -10 <= x2 <= 10
  0.5 <= x3 <= 10
  0 <= z <= 500

t=t0:
  x1 = 0
  x2 = 1
  x3 >= 0.5
  x4 = 0
  J = 0

t=tf:
  x1 = 0
  x3 >= 0.5
  x4 = 1
  J = 1

ini:
  x1 = 0
  x2 = 1
  x3 = 0.5
  x4 = linspace(0,1)
  z = 10

equ:
  x1´ == x2
  x2´ == -z*x1/sqr(x3)
  J´ == x3

obj:
  minimize z using nlp
