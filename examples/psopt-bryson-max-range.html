Bryson Maximum Range Problem

rem: Bryson Maximum Range Problem

Source: PSOPT - Optimal Control Solver, User Manual, Release 4 build 2015 

par:
  g = 1
  a = 0.5*g
  tf = 2

dyn:
  x y v   # state variables
  u{1:2}  # control variables
  
lim:
  -10 <= x <= 10
  -10 <= y <= 10
  -10 <= v <= 10
  -1 <= u{1:2} <= 1
  
t=t0:
  x = 0
  y = 0
  v = 0 
  
t=tf:
  y = 0.1
  
ini:
  x = initial(x)
  y = initial(y)
  v = initial(v)
  # To reproduce the results
  u1 = linspace(1,-1)
  u2 = sqrt(1-u1*u1)
  # Improved objective function
  # u1 = sqrt(0.5)
  # u2 = -sqrt(0.5)

equ:
  x´ == v*u1
  y´ == v*u2
  v´ == a-g*u2
  u1*u1 + u2*u2 == 1

obj: 
  maximize final(x) using nlp
