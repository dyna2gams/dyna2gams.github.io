Fuller Phenomenon

rem: Fuller Phenomenon

Find \(u\) over \(t \in [0,+\infty]\) to minimize: \[ J = \int_0^{+\infty} x_1^2 \ \mathrm{d}t \] subject to: \[ \dot{x}_1 = x_2 \\ \dot{x}_2 = u \\ x(t_0) = [10 \ 0 ] \\ x(t_f) = [0 \ 0 ] \\ |u| \leq 1 \]

Reference:

  1. U. Boscain, B. Piccoli, A Short Introduction to Optimal Control, Contrôle non linéaire et applications edited by Tewfik Sari, Travaux en cours, Hermann, Paris, LES COURS DU CIMPA, pp. 19-66, 2005.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010. 

var:
  tf

dyn:
  x1 x2	# state variables
  u	# control variable
  J

lim:
  1 <= tf <= 20
  -1 <= u <= 1

t=t0:
  x1 = 10
  x2 = 0
  J = 0

t=tf:
  x1 = 0
  x2 = 0

ini:
  tf = 10
  x1 = linspace(10,0)
  x2 = 0
  u = linspace(-1,1)

equ:
  x1´ == x2
  x2´ == u
  J´ == sqr(x1)

obj:
  minimize final(J) using nlp
