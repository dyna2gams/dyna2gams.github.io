Chebyshev Differential Equation

rem: Chebyshev Differential Equation

\[ (1-t^2) \frac{d^2 y}{d t^2} - x \frac{d y}{d t} + \alpha^2 y = 0 \]

for \(|t| \lt 1\).

The general solution is given in closed form by:

\[ y = a_1 \cos(\alpha \cos^{-1} t) + a_2 \sin(\alpha \cos^{-1} t) \] 

run:
  default-collocation-method = GL4/IG

par:
  alpha = 2
  tf = 2

dyn:
  t x1 x2

lim:
  -1 <= t <= 1

t=t0:
  t = -1
  x1 = 2

t=tf:
  x1 = 2

ini:
  t = 0.5
  x1 = 0
  x2 = 0

equ:
   t´ == 1
  x1´ == x2
  x2´ == (t*x2-alpha*alpha*x1)/(1-t*t)

obj:
  minimize 0 using nlp
