Flight Path Tracking

rem: Flight Path Tracking

Reference:

  1. J. Lygeros, Minimum Cost Optimal Control: An Application to Flight Level Tracking, Department of Engineering, University of Cambridge, Cambridge, UK.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010. 

par:
  cr2d = pi/180
  L = 65.3
  D = 3.18
  m = 160e3
  g = 9.81
  c = 6
  tf = 100

dyn:
  x1 x2 x3	# state variables
  u1 u2		# control variables
  J

lim:
  92 <= x1 <= 170
  -20*cr2d <= x2 <= 25*cr2d
  -150 <= x3 <= 150
  60e3 <= u1 <= 120e3
  -150 <= u2 <= 150

t=t0:
  x1 = 153.73
  x2 = 0
  x3 = 0
  J = 0

equ:
  x1´ == -D/m*x1^2-g*sin(x2)+u1/m
  x2´ == L/m*x1*(1-c*x2)-g*cos(x2)/x1+L*c/m*u2
  x3´ == x1*sin(x2)
  J´ == sqr(x3)

obj:
  minimize final(J) using nlp
