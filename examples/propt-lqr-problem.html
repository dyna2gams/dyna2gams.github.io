LQR Problem

rem: LQR Problem

Find \(u(t)\) over \(t \in [0,1]\) to minimize \[ J = \int_0^{1} (0.625 \ x^2 + 0.5 \ x \ u +0.5 \ u^2) \ \mathrm{d}t \] subject to: \[ \dot{x} = \frac{1}{2} x + u \\ x(0) = 1 \]

Reference:

  1. A. Schwartz, E. Polak and Y. Chen, RIOTS 95 A Matlab Toolbox for Solving Optimal Control Problems, Danville, CA 94526, USA.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010. 

par:
  tf = 1
  
dyn:
  x	# state variables
  u	# control variable
  J
  
lim:

t=t0:
  x = 1
  J = 0

ini:
  x = linspace(1,0)
  u = 0
  
equ:
  x´ == 0.5*x + u
  J´ == 0.625*sqr(x)+0.5*x*u+0.5*sqr(u)

obj:
  minimize final(J) using nlp
