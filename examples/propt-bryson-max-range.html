Bryson Maxrange

rem: Bryson Maxrange

Max range version of Bryson-Denham problem.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010 

par:
  g = 1
  a = 0.5*g
  tf = 2
  
dyn:
  x y v		# state variables
  u1 u2		# control variables

lim:
  -10 <= x <= 10
  -10 <= y <= 10
  -100 <= v <= 100
  -100 <= u1 <= 100
  -100 <= u2 <= 100

t=t0:
  x = 0
  y = 0
  v = 0 
  
t=tf:
  y = 0.1
  
ini:
  u1 = 1
  u2 = 1
  
equ:
  x´ == v*u1
  y´ == v*u2
  v´ == a - g*u2
  sqr(u1) + sqr(u2) == 1

obj: 
  maximize final(x) using nlp
