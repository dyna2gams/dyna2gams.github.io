System of ODEs with integral constraints

rem: System of ODEs with integral constraints

Solve the following system of ODE and integral constraint:

\[ \int_0^{1/2}\dot{x}^2(t)=p\\ 2 \lambda \ddot{x}(t) + \pi cos(\pi x(t)) = 0\\ x(0)=0, x(1/2)=1/2 \]

where \(p\) is a given constant, and \(\lambda\) is a constant that has to be determined.

Reference:

  1. https://math.stackexchange.com/questions/3042833/ 

par:
  p = 0.6
  tf = 0.5

var:
  lambda

dyn:
  x{1:3}

t=t0:
  x1 = 0
  x3 = 0
  
t=tf:
  x1 = 0.5
  x3 = p

equ:
  x1´ == x2
  x2´ == -0.5*pi*cos(pi*x1)/lambda
  x3´ == sqr(x2)
  
ini:
  lambda = 1

obj:
  minimize initial(x1) using nlp	# dummy objective
