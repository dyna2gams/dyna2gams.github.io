Hanging Chain (COPS Problem 4)

rem: Hanging Chain (COPS Problem 4)

See: COPS: Large-Scale Optimization Problems 

par:
  a = 1
  b = 3
  Lp = 4
  tf = 1

dyn: x L E u

lim:
  0 <= x <= 10
  0 <= L <= 10
  0 <= E <= 10
  -10 <= u <= 20

t=t0:
  x = a
  E = 0
  L = 0

t=tf:
  x = b
  L = Lp

ini:
  x = (a+b)/2
  L = 5
  E = 5
  u = 0

equ:
  x´ == u
  E´ == x*sqrt(1+u*u)
  L´ == sqrt(1+u*u)

obj:
  minimize final(E) using nlp
