Food Sterilization

rem: Food Sterilization

Simplified version of the sterilization problem considered in the paper here after.

Reference:

  1. Z.S. Chalabi, L.G. van Willigenburg, G. van Straten, Robust optimal receding horizon control of the thermal sterilization of canned food, Journal of Food Engineering, 40:207-218, 1999.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010. 

par:
  tf = 1500
  pv{1:8} = {0.01 0.005 0.01 20 10 121.11 25.56 121.11}

dyn:
  x{1:4}	# state variables
  u	# control variable

lim:
  0 <= x{1:4} <= 300 
  0 <= u <= 50

t=t0:
  x{1:4} = {20 20 0 0}

ini:
  x{1:4} = linspace({20 20 0 0},{40 0 100 300})
  u = 50

equ:
  x1´ == pv1*(x2-x1)
  x2´ == pv2*(pv4-x2)+pv3*u
  x3´ == exp(log(10)/pv5*(x1-pv6))
  x4´ == exp(log(10)/pv7*(x1-pv8))

obj:
  minimize final(x4)+sqr(final(x3)-100)+sqr(final(x1)-40) using nlp
