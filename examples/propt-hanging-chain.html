Hanging Chain

rem: Hanging Chain

Find \(x(t)\) over \(t \in [0,1]\) to minimize \[ J = \int_0^{1} x \sqrt{1+(\dot{x})^2} \ \mathrm{d}t \] subject to: \[ \int_0^{1} \sqrt{1+(\dot{x})^2} \ \mathrm{d}t = 4 \\ x_0 = 1 \\ x_1 = 3 \]

Reference:

  1. E. D. Dolan, J. J. Moré and T. S. Munson, Benchmarking Optimization Software with COPS 3.0, Argonne National Laboratory, Technical Report ANL/MCS-TM-273, February 2004.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010 

par:
  a = 1
  b = 3
  tf = 1
  
dyn:
  x	# state variables
  u
  J K

lim:
  -10 <= u <= 20
  
t=t0:
  x = a
  J = 0
  K = 0

t=tf:
  x = b
  K = 4
  
ini:
  x = (a+b)/2
  u = 0

equ:
  x´ == u
  J´ == x*sqrt(1+sqr(u))
  K´ == sqrt(1+sqr(u))  

obj:
  minimize final(J) using nlp
