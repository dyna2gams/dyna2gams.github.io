Ball and beam problem

rem: Ball and beam problem

The ball-and-beam problem is a commonly recurring nonlinear control application used by authors to demonstrate the effectiveness of various control methods.

Reference:

  1. M. Palmer and X. Feng, Optimal control solutions for the ball-and-beam problem, IFAC Proceedings, 32(2):2911-2914, 1999. 

par:
  alpha = 0.7143
  beta = 9.81
  tf = 6

dyn:
  x{1:4}
  u
  J

t=t0:
  x{1:4} = {2 0 0 0}
  J = 0

t=tf:
  x{1:4} = 0

equ:
  x1´ == x2
  x2´ == alpha*(x1*x4*x4 - beta*sin(x3))
  x3´ == x4
  x4´ == u
  J´ == x1*x1 + x2*x2 + x3*x3 + x4*x4 + u*u

obj:
  minimize final(J) using nlp
