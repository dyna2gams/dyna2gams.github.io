Coulomb Friction

rem: Coulomb Friction

Find \(u\) over \(t \in [0,t_f]\) to minimize \[ J = t_f \] subject to: \[ m_1 \ \ddot{q}_1 = (-k_1-k_2) \ q_1 + k_2 \ q_2 - \mu \ \mathrm{sgn}(\dot{q}_1) + u_1 \\ m_2 \ \ddot{q}_2 = k_2 \ q_1 - k_2 \ q_2 - \mu \ \mathrm{sgn}(\dot{q}_2) + u_2 \\ q_{1:2}(0) = [0 \ 0] \\ \dot{q}_{1:2}(0) = [-1 \ -2] \\ q_{1:2}(t_f) = [1 \ 2] \\ \dot{q}_{1:2}(t_f) = [0 \ 0] \\ -4 \leq u_{1:2} \leq 4 \\ k_{1:2} = [0.95 \ \ 0.85] \\ m_{1:2} = [1.1 \ \ 1.2] \\ \mu = 1 \]

Reference:

  1. B.J. Driessen, N. Sadegh, Minimum-Time Control of Systems With Coulomb Friction: Near Global Optima Via Mixed Integer Linear Programming, Optimal Control Applications and Methods, 22(2), March-April 2001.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010. 

par:
  k{1:2} = {0.95 0.85}
  m{1:2} = {1.1 1.2}
  mu = 1

var:
  tf

dyn:
  q1 q1dot q2 q2dot	# state variables
  u1 u2	# control variables

lim:
  1.8 <= tf <= 4
  -4 <= u1 <= 4
  -4 <= u2 <= 4

t=t0:
  q1 = 0
  q1dot = -1
  q2 = 0
  q2dot = -2

t=tf:
  q1 = 1
  q1dot = 0
  q2 = 2
  q2dot = 0

ini:
  q1 = linspace(0,1)
  q1dot = linspace(-1,0)
  q2 = linspace(0,2)
  q2dot = linspace(-2,0)
  u1 = 1
  u2 = 1

equ:
  q1´ == q1dot
  m1*q1dot´ == (-k1-k2)*q1+k2*q2-mu*sign(q1dot)+u1
  q2´ == q2dot
  m2*q2dot´ == k2*q1-k2*q2-mu*sign(q2dot)+u2

obj:
  minimize tf using dnlp with snopt
