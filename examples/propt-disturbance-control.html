rem: Disturbance Control

Algorithm of Acting Optimal Controller.

Find \(u\) over \(t \in [0,25]\) to minimize: \[ J = 0 \] subject to: \[ \dot{x}_1 = x_3 \\ \dot{x}_2 = x_4 \\ \dot{x}_3 = -x_1+x_2+u \\ \dot{x}_4 = 0.1 \ x_1-1.02 \ x_2+0.3 \ \sin(4t) \ (t<9.75) \\ x(t_0) = [0 \ \ 0 \ \ 2 \ \ 1] \\ x(t_f) = [0 \ \ 0 \ \ 0 \ \ 0] \\ 0 \leq u \leq 1 \]

Reference:

  1. F.M. Kirillova, Optimal On-Line Control and Classical Regulation Problem, Institute of Mathematics National Academy of Sciences of Belarus.

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010.
