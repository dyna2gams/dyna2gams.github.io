Linear Problem with Bang Bang Control

rem: Linear Problem with Bang Bang Control

Find \(u\) over \(t \in [0,1]\) to minimize \[ J = \int_{0}^{1} (-6 x_1-12 x_2+3 u_1+u_2) \ \mathrm{d}t \] subject to: \[ \dot{x}_1 = u_2 \\ \dot{x}_2 = -x_1+u_1 \\ x_1(0) = 1 \\ x_2(0) = 0 \\ |u| \leq 10 \]

Reference:

  1. L. S. Jennings, M. E. Fisher, K. L. Teo, C. J. Goh, MISER3 Optimal Control Software: Theory and User Manual Version 2.0, Department of Mathematics, The University of Western Australia, Nedlands, WA 6907, Australia (1997).

Source: PROPT - Matlab Optimal Control Software, User Manual, April 26, 2010 

par:
  tf = 1

dyn:
  x{1:2}	# state variables
  u{1:2}:integer	# control variables
  J

lim:
  -10 <= x1 <= 10
  -10 <= x2 <= 10
  -10 <= u1 <= 10
  -10 <= u2 <= 10

t=t0:
  x1 = 1
  x2 = 0
  J = 0

ini:
  x1 = 1
  x2 = 0
  u1 = 0
  u2 = 0

exp:

equ:
  x1´ == u2
  x2´ == -x1+u1
  J´ == -6*x1-12*x2+3*u1+u2

obj:
  minimize final(J) using mip
